Данный проект реализован на playwright + bs4. С помощью playwright динамически собираются данные по всем точкам по карте, при выборе города вызывается php файл, который возвращает всю информацию в json формате. После чего они уже приводятся к нормализированному типу как в ТЗ. Также, от себя я добавил сбор ссылок на каждый билборд, для удобства поиска и последующего заказа.

**Установка зависимостей:**  
`pip install -r requirements.txt`  
`playwright install chromium`

**Запуск проекта:**  
Для того, чтобы запустить проект, нужно лишь запустить `main.py`. После запуска файла в консоли будет выведен список всех городов: можно выбрать один отдельно или все города разом в общий файл. После выбора города предлагается формат данных, удобный для пользователя.
