# testcase_parser
Данный проект реализован на playwright + bs4. С помощью playwright динамически собираются данные по всем точкам по карте, при выборе города вызывается php файл, который возвращает всю информацию в json формате. После чего они уже приводятся к нормализированному типу как в ТЗ.
установка зависимостей - pip install -r requirements.txt
Для того, чтобы запустить проект, нужно лишь запустить main.py. После запуска файла, в консоли будет выведет список всех городов, можно выбрать один отдельно, а можно и все города разом в общщий файл. После выбора города, дается выбор удобного формата для пользователя, в котором ему нужно получить данные.
